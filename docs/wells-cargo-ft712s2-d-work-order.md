## Build-relevant notes vs. the plan
- **2"×4" main rails confirmed** — `fastrac-specs.md` had this as a low-confidence derivation; the factory order verifies it. E-track floor bolts land in 2"×4" steel.
- **Single 3.5K electric-brake axle** confirmed (matches "electric drum brakes").
- **4× factory 5,000 lb D-rings + 3/4" PlexCore floor** — tie-down anchors already on the unit, in addition to the planned E-track.
- **Discrepancy — color:** work order says **Silverfrost** (silver/grey); the context dump says white.
- **Discrepancy — side door:** factory unit has a **32"×72" PT (personnel) door, RH hinge** — not the "RV-style door" noted earlier. An RV-style door would be an aftermarket swap.
